# 밥머니 (bobmoney)

식비만 간단히 기록하고, 지난주와 이번 주 소비를 비교해 다음 행동을 정할 수 있는 개인용 웹앱입니다.

## 주요 기능

- 월 식비 예산 설정(기본 600,000원)
- 날짜·금액·종류·만족도·메모 기록
- 월 사용액, 남은 예산, 하루 평균, 남은 하루 한도 표시
- 지난주/이번 주 지출 및 주간 가이드 비교
- 월별 주차 지출 그래프
- 카테고리별 합계와 기록 수정·삭제
- JSON 백업 및 복원
- 브라우저 `localStorage`에 데이터 저장

## 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

로컬 서버를 사용하려면 저장소 폴더에서 다음 중 하나를 실행하세요.

```bash
python -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000`으로 접속합니다.

## GitHub Pages

저장소의 **Settings → Pages**에서 배포 소스를 `Deploy from a branch`, 브랜치를 `main / root`로 선택하면 웹에서 사용할 수 있습니다.

## 데이터 주의사항

기록은 서버가 아니라 현재 브라우저에만 저장됩니다. 브라우저 데이터를 지우거나 기기를 변경하기 전에는 설정 메뉴의 **백업 파일 저장**을 사용하세요.
